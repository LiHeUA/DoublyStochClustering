This set of m-files demonstrates the Doubly Stochastic Clustering method proposed in [1]. DS clustering replaces the Euclidean distance matrix in standard clustering with its nearest doubly stochastic matrix.

To run the demo, simply start Matlab and run demo.m

[1] L. He, H. Zhu, T. Zhang, Y. Guan and H. Zhang, K-means via Doubly Stochastic Scaling, submitted to TSP

Please cite this paper if you use the code to generate your results.
